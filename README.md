
# 👻 Prizrak Box Config

Простой конфиг для Mihomo / Prizrak Box. Настроен на обход блокировок, работу игр, TeamSpeak и Discord.

## 🚀 Инструкция по установке

### Шаг 1: Получение кода
1. Откройте файл `prizrak.yaml` в этом репозитории.
2. Нажмите кнопку **«Raw»** в верхней части окна.
3. Выделите весь текст (**Ctrl + A**) и скопируйте его (**Ctrl + C**).

### Шаг 2: Настройка в Prizrak Box
1. Откройте приложение **Prizrak Box**.
2. Перейдите в раздел **«Правило»** (левое меню).
3. Нажмите кнопку **«Единые правила»** в верхней части окна.
4. Нажмите кнопку **«Добавить»** над полем ввода.
5. Вставьте скопированный код и нажмите **«Сохранить»**.
6. **Важно:** Убедитесь, что ползунок **ВЫКЛ / ВКЛ** над полем кода переведен в положение **ВКЛ**.

## 🛠 Состав правил
Конфиг включает автоматическую маршрутизацию для следующих сервисов и протоколов:

* **Видео и стриминг:** YouTube, Twitch (AdBlock)
* **Игровые сервисы:** Steam, Osu!, Clash Royale
* **Голосовая связь:** TeamSpeak, Discord (включая Voice IPs)
* **Мессенджеры:** Telegram (включая AyuGram)
* **Музыка:** Spotify
* **Нейросети (AI):** ChatGPT, Google Gemini (DeepMind)
* **Системные ресурсы:** WSL (Windows Subsystem for Linux), Cloudflare
* **Удаленное управление:** AnyDesk, RustDesk, TeamViewer, Tailscale, Wireguard, Netbird
* **Региональные ресурсы:** Прямой доступ для RU/BY сегмента (.ru, .su, .by, Yandex, Mail.ru, VK, OK, Gosuslugi)
* **Безопасность и фильтрация:** Блокировка рекламы (Oisd Big), блокировка QUIC, обход блокировок (Re-filter, Ru-inside)
* **Служебное:** Speedtest, IP-чекеры, Torrent-клиенты и трекеры (Direct)

## ⚠️ Важное примечание
На главной панели слева доступны два режима работы: **Прокси** и **Tun**. 

Для корректной работы этого конфига **рекомендуется использовать режим Tun**. Это позволяет системе захватывать трафик игр и программ (TeamSpeak, Discord) на уровне сетевого драйвера, что исключает возникновение конфликтов и ошибок подключения. Убедитесь, что тумблер **Tun** находится в активном состоянии.
